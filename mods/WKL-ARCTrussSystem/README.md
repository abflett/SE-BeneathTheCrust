# Working Knowledge Layer - ARC Truss System

Compatibility layer for using ARC Truss System blocks with Working Knowledge progression.

Version: `1.0.0`

Load order:

1. Working Knowledge
2. ARC Truss System
3. Working Knowledge Layer - ARC Truss System

This mod provides vanilla research definitions for ARC Truss System blocks and maps those blocks into Working Knowledge schematic families.
