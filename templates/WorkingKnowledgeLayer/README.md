# {{LAYER_NAME}}

Compatibility layer for using **{{SOURCE_MOD_NAME}}** blocks with **Working Knowledge** progression.

This folder is a template. Replace the `{{...}}` placeholders, remove `.template` suffixes, and keep the finished mod as a normal Space Engineers mod root.

## Files

- `modinfo.sbc` - Space Engineers mod metadata.
- `Data/ResearchBlocks.sbc` - vanilla progression definitions for each supported block.
- `Data/WorkingKnowledge/block_mappings.txt` - Working Knowledge schematic-family mappings.

## Typical Workshop Requirements

- Working Knowledge
- {{SOURCE_MOD_NAME}}

If publishing through Steam Workshop, set those as required items on the Workshop page.
