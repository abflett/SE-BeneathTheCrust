# Templates

Reusable project templates live here. These are not loadable Space Engineers mod roots and should not be deployed directly.

## WorkingKnowledgeLayer

[WorkingKnowledgeLayer/](WorkingKnowledgeLayer/README.md) is a starter structure for creating a small compatibility layer that maps another block mod into Working Knowledge schematic families.

See [Working Knowledge layer authoring](../docs/WorkingKnowledge/layer_authoring.md) for the full workflow.
