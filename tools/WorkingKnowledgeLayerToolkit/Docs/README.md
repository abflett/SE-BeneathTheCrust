# Toolkit Docs

Start with [Quick Start](../QUICKSTART.md) if you want the script to generate a layer.

Use these pages when you want to review or edit the generated files:

- [Editing Generated Layers](editing_generated_layers.md) - plain-language guide to the files most users may edit.
- [Mapping Format](mapping_format.md) - exact `block_mappings.txt` syntax.
- [Schematic Groups](schematic_groups.md) - built-in IDs and the versioned custom-group format.
- [Manual Layer Authoring](manual_layer_authoring.md) - how to build a layer without the script.
- [Troubleshooting](troubleshooting.md) - common scan and load-order issues.

The toolkit docs are Markdown files. They read best on GitHub, but any text editor can open them.
