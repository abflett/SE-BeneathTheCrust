[h1]Working Knowledge Layer - ARC Truss System[/h1]
[h3]Compatibility support for using ARC Truss System with Working Knowledge progression.[/h3]

This small layer mod connects [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3701176691]ARC Truss System[/url] blocks to [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3758066250]Working Knowledge[/url].

With this layer loaded, ARC Truss System blocks are no longer outside the Working Knowledge progression model. They can be assigned to schematic families, gated by research, practiced through Proficiency, and handled by the normal Working Knowledge welding, grinding, repair, and salvage systems.

For normal Steam Workshop use, add this layer mod to your world and Steam should resolve the required mods automatically.

[h2]What It Does[/h2]
[list]
[*]Adds vanilla research block definitions for ARC Truss System blocks.
[*]Maps ARC truss structure blocks to [b]Industrial Structure Schematics[/b].
[*]Maps the ARC truss light to [b]Interior Lighting Schematics[/b].
[*]Lets Working Knowledge treat those blocks like normal researched/proficient technology.
[/list]

[h2]Workshop Requirements[/h2]
[list]
[*][url=https://steamcommunity.com/sharedfiles/filedetails/?id=3758066250]Working Knowledge[/url]
[*][url=https://steamcommunity.com/sharedfiles/filedetails/?id=3701176691]ARC Truss System[/url]
[/list]

Working Knowledge also depends on Text HUD API for its progress overlay, so that dependency may be pulled in through Working Knowledge.

[h2]Manual Or Offline Setup[/h2]
If you are copying mods locally, using offline files, or building a manual modpack, include:

[code]Working Knowledge
ARC Truss System
Working Knowledge Layer - ARC Truss System[/code]

The layer is safest after both parent mods in a manual list, but normal Steam dependency resolution should handle this for Workshop worlds.

[h2]Notes[/h2]
This mod does not include ARC Truss System and does not add new blocks, models, recipes, or balance rules. It is only a compatibility layer.

If ARC Truss System changes its block subtype IDs in a future update, this layer may need to be updated.

[h2]Status[/h2]
Version 1.0.1 is the current stable layer release.
